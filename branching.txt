1.branching steps:-
 - git branch --to check the exist branch
 - git branch <branch- name> -- to create new branch 
 - git checkout <branch-name> -- to switch from one branch to another branch





------------------- update ----------------------
  
  - git add .  ----- to add updated file on the index stage 
  - git commit -m "message" ---- to commit or move from index stage to local repository.
  - git push -u origin <branch-name> ---- push to the remote repository.
